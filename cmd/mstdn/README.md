# mstdn

command line tool for mstdn.jp

## Usage

```
Usage of mstdn:
  -S	streaming public
  -ff string
    	post utf-8 string from a file("-" means STDIN)
  -t string
    	toot text
```

## Installation

```
$ go get github.com/mattn/go-mastodon/cmd/mstdn
```
